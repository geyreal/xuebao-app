# 📚 学伴小书包

> 智慧学习，快乐成长

校内学习资源管理与打卡应用，基于 **uni-app (Vue 3 + TypeScript)** 构建，一套代码覆盖 Android / iOS / H5。

## ✨ 功能特性

### 学生端
- 📖 **学科中心** — 按语文/数学/英语/综合浏览资源，支持搜索和分页
- ✅ **打卡中心** — 一键打卡、打卡日历、连续天数统计
- ❤️ **我的收藏** — 收藏喜欢的资源，快速找回
- 📊 **学习记录** — 记录学习进度

### 教师端
- 📤 **上传资源** — 支持 PDF、Word、图片、音频
- 📋 **发布打卡任务** — 设定截止时间，查看打卡名单
- 🎓 **教师工作台** — 管理资源和任务

### 系统
- 🔐 **角色权限** — 学生/教师/管理员三级权限
- 🏫 **班级隔离** — 数据按班级自动过滤，确保安全隔离
- 📢 **公告系统** — 全校通知下发

## 🛠️ 技术栈

| 层次 | 技术 |
|------|------|
| 前端框架 | uni-app + Vue 3 + TypeScript |
| 构建工具 | Vite 5 |
| 状态管理 | Vue 3 原生 reactive（无第三方依赖） |
| 后端 | Go (Gin) — *待开发* |
| 数据库 | MySQL 8.0 — *待开发* |
| 文件存储 | 阿里云 OSS — *待开发* |

> 💡 前端坚持**依赖精简原则**，不引入 Pinia、vue-i18n 等第三方库，使用 Vue 3 原生能力实现。

## 🚀 快速开始

```bash
# 安装依赖
cd xuebao-app
npm install

# 启动 H5 开发服务器
npm run dev:h5

# 编译微信小程序
npm run dev:mp-weixin

# 构建 H5 生产包
npm run build:h5
```

启动后访问 `http://localhost:5173/`，输入任意账号密码登录（T 开头为教师端）。
